## Описание задания:
#### Создать мини-приложение для отображения списка объектов недвижимости с возможностью переключения валюты.

- Стек:
- Next.js 16
- TypeScript
- React
- Tailwind CSS
- Shadcn UI

Задача.

1. Страница со списком объектов.
   Необходимо реализовать страницу с 5–10 карточками недвижимости (данные можно захардкодить).
   В каждой карточке должны отображаться: название объекта, фото-placeholder, цена и площадь.
   Базовая цена должна храниться в THB (тайский бат).

2. Переключатель валюты.
   В header необходимо разместить dropdown с выбором валюты: THB, USD, EUR, RUB.
   Выбор пользователя должен сохраняться между сессиями.
   Выбранная валюта должна корректно отображаться при SSR, без визуального мерцания.

3. Компонент отображения цены.
   Компонент должен принимать сумму в THB.
   Цена должна отображаться в выбранной пользователем валюте.
   Необходимо корректное форматирование чисел и символов валют.

4. Курсы валют (статичные).
   THB — 1
   USD — 0.0286
   EUR — 0.0263
   RUB — 2.57

#### Технические требования.
- Использовать Next.js App Router.
- TypeScript в strict mode.
- Tailwind CSS и Shadcn UI.
- Код писать на английском языке.
- UI-тексты — на русском языке.
